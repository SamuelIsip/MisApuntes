Objeto Navigator & Screen

Navigator

El objeto navigator identifica las características del navegador sobre el que ejecutamos la aplicación web escrita en JavaScript. Permite conocer datos como el tipo de navegador su versión o el sistema operativo del usuario. Se usa para obtener esta información y tomar decisiones del tipo de código a ejecutar. Esto es debido a que no todos los navegadores se comportan igual ante el mismo código

Métodos Navigator

Screen

El objeto screen muestra las características de la pantalla del usuario como dimensiones o color Posee atributos de lectura pero ningún método La altura y anchura suelen ser mayores que la altura y anchura disponibles pues el sistema operativo utiliza una parte de la pantalla Su utilidad es informar sobre datos de la pantalla a diseñadores web para adaptar los diseños adecuadamente antes de cargarlos
