Objeto History & Location

History

Este objeto hace referencia al historial del navegador y por tanto a las páginas visitadas. Este historial se almacena en un array y usando los atributos y métodos podemos avanzar o retroceder a lo largo del historial. Aunque podemos ir de una página a otra de las visitadas no podremo ver la dirección de la página para conservar la privacidad del usuario. Sus métodos son el acceso a los documento.

Location

El objeto location referencia la barra de direccio-nes del navegador web. Su utilidad es manipular la URL del documento. Con este objeto podemos consultar y manipular cada una de las partes de la dirección web. Puede realizar las mismas operaciones que realiza el navegador con las URL's tales como cargar una página, refrescarla u obtener datos como las variables pasadas por parámetro.
